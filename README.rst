
===============================
Tools for Network Testing
===============================

**msend**
::

 msend -1 239.1.1.100 3300 20 192.168.167.1
  group : 239.1.1.100 
  port  : 3300
  ttl   : 20
  interface : 192.168.167.1 is the ip address of one interface

  
**mdump**
::
 mdump -Q 0 239.1.1.100 3300 192.168.167.1
